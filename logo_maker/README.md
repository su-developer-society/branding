# `logo_maker`
This tool was created so that members of the developer society can easily
create the logo for the society, as well as logos for other events that the
society organises, like hackathons.

Conribution in the form of documentation or tool improvement is welcome!

## Usage
To use this tool clone the repository, and navigate to the directory where this
file is stored. In your terminal, run:
```
npm install
```
After packages have been installed, use the tool by running:
```
node generate_logo.js
```
